# External Inputs

**Status:** DISCUSSION_DRAFT  
**Canonical status:** Non-canonical  
**Purpose:** Archive of external expert inputs relevant to RABA inquiries  

## Purpose

This directory preserves external inputs from:

- LinkedIn comments;
- GitHub issues and discussions;
- direct messages or email;
- external review cards;
- third-party trace examples;
- specialist feedback;
- partner conversations.

## Boundary

External input is not approval.

A public comment, expert opinion, AI-generated review, or third-party trace may inform RABA, but it does not automatically become RABA architecture.

## Handling Rule

For each external input, record:

- source type;
- author / organization if public;
- date;
- permission / quotation status;
- summary;
- relevant inquiry;
- RABA interpretation;
- accepted / rejected / deferred points;
- Human Owner review status.

## Quotation Rule

Prefer paraphrase unless explicit permission to quote is available or the source is already public and citation is appropriate.

Do not treat external agreement as final validation.
