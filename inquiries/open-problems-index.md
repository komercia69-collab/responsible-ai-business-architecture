## Purpose

This index tracks open problem questions that RABA is actively investigating or preserving for later review.

Each problem should link to:

- relevant legal or standards sources;
- research notes;
- external expert inputs;
- resolution attempts;
- current RABA interpretation;
- adoption status.

## Status Values

| Status | Meaning |
|---|---|
| Open | Problem identified, no stable resolution attempt yet |
| Under review | Sources and external inputs are being collected |
| Resolution attempt | RABA has a proposed approach, not yet accepted |
| Concept draft exists | A concept document exists but is non-canonical |
| Governance draft exists | A governance document exists but is non-canonical |
| Deferred | Preserved for later review, not active |
| Accepted | Human Owner approved the result into project architecture |
| Rejected | Human Owner rejected the proposed resolution |

## Adoption Rule

No open problem becomes canonical RABA architecture through discussion alone.

External agreement, multi-AI convergence, LinkedIn consensus or GitHub discussion is input, not approval.

Adoption requires Human Owner review and explicit confirmation.
