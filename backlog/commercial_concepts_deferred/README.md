# Deferred Commercial Concepts

**Status:** Deferred backlog  
**Canonical status:** Non-canonical  
**Commercial status:** Not active  
**Approval status:** Human Owner review required before any commercial action  

> Multi-AI agreement is not approval.

## Purpose

This directory preserves commercial concepts that were generated during RABA development but explicitly deferred by the Human Owner.

The purpose is to avoid losing useful thinking while preventing premature commercial activation.

## What this directory is

This is a backlog archive for future consideration.

It may contain summaries of:

- pilot offer ideas;
- positioning statements;
- integration candidates;
- revenue sustainability models;
- commercial product hypotheses;
- potential future RABA Compliance Profile directions.

## What this directory is not

This directory is not:

- an active commercial offer;
- a pilot proposal;
- a compliance product;
- a pricing page;
- a client commitment;
- a legal or regulatory guarantee;
- a live business track.

## Active Boundary

RABA does not currently claim that it can certify EU AI Act compliance.

RABA is an open-source responsibility architecture framework. Any future commercial compliance profile would require separate review, legal boundary definition, Human Owner approval and explicit activation.

## Reopening Criteria

Commercial concepts may be reopened only when:

1. the open-source RABA core is stable enough to support external use;
2. framework boundaries are clearly documented;
3. regulatory mapping drafts have been reviewed;
4. legal/compliance review has been obtained for any external offer;
5. the Human Owner explicitly confirms commercial activation.
