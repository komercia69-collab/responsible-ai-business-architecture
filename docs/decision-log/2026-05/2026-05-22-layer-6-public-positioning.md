## Positioning Rule

“Layer 6” is a communication frame.

It means:

```text
Existing AI architecture layers define model, runtime, memory, tools and routing.
RABA adds a governance layer for responsibility, authorization, escalation and auditability.
```

It does not mean:

```text
RABA replaces existing runtime, memory, extension, routing or safety architecture.
```
